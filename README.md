# Pradnya Portfolio

A personal portfolio website showcasing my projects, technical skills, experience, and achievements.

## Live Demo

🌐 Portfolio Website: https://pradnyajprotfolio.vercel.app/

## Features

- Responsive design
- Projects showcase
- Experience section
- Skills section
- Contact information
- Resume download

## Tech Stack

- HTML
- Tailwind CSS
- JavaScript

## Projects Highlighted

### SafePath – Mumbai Waterlogging
A flood-awareness platform that visualizes waterlogging-prone areas and recommends safer travel routes using geospatial data.

### QuickSend – Email Marketing Platform
A MERN-based email marketing platform supporting bulk email campaigns, scheduling, audience segmentation, and customizable templates.

### FoodBridge
A MERN-based platform connecting hotels, NGOs, and volunteers to streamline surplus food distribution and reduce food wastage.
